# In-House Goal Setting & Tracking Portal

A role-based web portal for employee goal creation, manager approval, quarterly achievement tracking, HR reporting, and audit management. This project was built for **AtomQuest Hackathon 1.0**.

## Project Description

The In-House Goal Setting & Tracking Portal is a web-based employee performance goal management system. It helps organizations replace manual goal tracking through spreadsheets, emails, and offline reviews with a structured digital platform. Employees can create and submit goal sheets, managers can review and approve goals, employees can update quarterly achievements, and HR/Admin can monitor progress through dashboards, reports, and audit logs.

The system supports the complete lifecycle of goals, including goal creation, validation, approval, locking, quarterly check-ins, achievement tracking, reporting, shared KPIs, and audit history.
